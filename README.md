# 日期和时间计算工具

## 简介
这是一个基于React的日期和时间计算工具，提供了以下功能：
1. 计算从现在起或过去的时间差值（秒、分钟、小时、天、周、月、年）。
2. 查询某日期是星期几。
3. 计算两个时间点之间的小时数。
4. 计算两个日期之间的工作日数量。
5. 对给定时间进行加减小时的运算。

## 使用方法
1. 克隆本仓库到本地。
2. 在项目根目录下执行 `npm install` 安装依赖。
3. 执行 `npm start` 启动项目，在浏览器中访问 `http://localhost:3000` 即可使用。
